Encodings Class Reference
[Unicode core]

Detailed Description

Encodings information.

  • All codepage definitions are read from list of ANSI -> Unicode associative files. These codepage files could be taken from www.unicode.org too.
  • Two-stage tables are used to access Unicode -> ANSI transformation.
