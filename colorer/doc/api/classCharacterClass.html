CharacterClass Class Reference
[Unicode core]

Detailed Description

Character classes store implementation.

  • CharacterClass allows to store enumerations of characters in compact form (two-stage bit-field tables).
  • This class supports logical operations over it's instances, char category -> enumeration conversion.
